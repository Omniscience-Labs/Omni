## Setup

<Steps>
  <Step title="Connect Google Analytics">
    In Integrations, click **Connect Google Analytics**.

    {/* 
      ðŸ“¸ MEDIA PLACEHOLDER: Google Analytics Connection
      - GA integration card
      - Connect button
      - Capabilities overview
      - Size: 1200x800px
    */}
  </Step>

  <Step title="Google OAuth">
    Authorize access:
    - Sign in with Google account
    - Select Analytics properties to access
    - Grant read permissions
    - Click **Allow**

    <Info>
      Omni requests read-only access to Analytics data - cannot modify your GA settings or data.
    </Info>
  </Step>

  <Step title="Select Properties">
    Choose which GA properties to access:
    - Website properties
    - Mobile app properties
    - Multiple properties supported

    {/* 
      ðŸ“¸ MEDIA PLACEHOLDER: Property Selection
      - List of available GA properties
      - Checkboxes for selection
      - Property IDs shown
      - Size: 1400x800px
    */}

    Different agents can access different properties.
  </Step>

  <Step title="Test Integration">
    Verify data access:
    ```
    "Show me website traffic for the last 7 days"
    "What are my top pages this month?"
    ```

    <Check>
      Google Analytics connected! Data extraction enabled.
    </Check>
  </Step>
</Steps>

## Automated Reporting

<Steps>
  <Step title="Define Report Parameters">
    ```
    Report Spec:
    - Metrics: Sessions, Users, Conversions
    - Dimensions: Source, Medium, Campaign
    - Date Range: Last 7 days
    - Comparison: Previous 7 days
    - Segments: Mobile vs Desktop
    ```
  </Step>

  <Step title="Pull Data">
    Agent queries Google Analytics API:
    - Authenticates automatically
    - Retrieves specified metrics
    - Applies filters and segments
    - Handles pagination

    {/* 
      ðŸ“¸ MEDIA PLACEHOLDER: Data Extraction
      - Progress indicator
      - Metrics being pulled
      - API calls visualized
      - Data preview
      - Size: 1600x900px
    */}
  </Step>

  <Step title="Analyze & Visualize">
    ```
    "Create visualizations:
    - Traffic trend line chart
    - Source comparison pie chart
    - Conversion funnel
    - Top pages bar chart
    - Device breakdown"
    ```

    Professional charts auto-generated.
  </Step>

  <Step title="Generate Report">
    ```
    "Compile into executive report:
    - Summary of key metrics
    - Week-over-week changes
    - Notable trends
    - Recommendations
    - All visualizations included
    
    Export as PDF"
    ```

    <Check>
      Complete analytics report ready!
    </Check>
  </Step>

  <Step title="Distribute">
    ```
    "Send report to:
    - Email: marketing@company.com
    - Slack: #analytics channel
    - Save to: Dropbox /Reports/"
    ```

    Automated distribution.
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="No Data Returned" icon="database">
    **Check**:
    - Date range has data
    - Property ID correct
    - Filters not too restrictive
    - Sufficient permissions granted

    Verify in GA interface first.
  </Accordion>

  <Accordion title="Metrics Don't Match GA UI" icon="not-equal">
    **Possible Reasons**:
    - Different date ranges
    - Timezone differences
    - Sampling applied (large datasets)
    - Filters or segments different

    Compare exact parameters.
  </Accordion>
</AccordionGroup>

## Limitations

<Info>
  **Slack Rate Limits**:
  - 1 message per second per channel
  - 100 messages per minute workspace-wide
  - File uploads: 20 per minute

  Omni automatically manages rate limiting for you.
</Info>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Bot Not Responding" icon="robot">
    **Check**:
    - Bot is invited to channel
    - Mentioned correctly (@omni)
    - Integration is active
    - Agent has Slack enabled

    Re-invite bot to channel if needed.
  </Accordion>

  <Accordion title="Can't Access Channel" icon="lock">
    **For Private Channels**:
    - Bot must be explicitly invited
    - Use `/invite @omni` in channel
    - Admin may need to approve

    Public channels accessible automatically.
  </Accordion>

  <Accordion title="Messages Not Sending" icon="exclamation">
    **Possible Causes**:
    - Rate limit exceeded
    - Channel archived
    - Bot permissions insufficient
    - Connection expired

    Check integration status and re-authenticate.
  </Accordion>
</AccordionGroup>

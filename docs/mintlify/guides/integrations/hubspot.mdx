## Setup

<Steps>
  <Step title="Connect HubSpot">
    Navigate to **Integrations** and click **Connect HubSpot**.

    {/* 
      ðŸ“¸ MEDIA PLACEHOLDER: HubSpot Connection
      - HubSpot integration card with Connect button
      - Capabilities overview
      - OAuth flow preview
      - Size: 1200x800px
    */}
  </Step>

  <Step title="HubSpot OAuth">
    Authorize Omni access:
    - Select HubSpot account (or portal)
    - Review requested scopes
    - Click **Grant Access**

    <Info>
      Permissions include: Contacts, Companies, Deals, Marketing Emails, Forms, Workflows, and Analytics access.
    </Info>
  </Step>

  <Step title="Configure Sync Settings">
    **Optional Configuration**:
    - Default pipeline for deal creation
    - Contact owner assignment rules
    - Custom property mappings
    - Notification preferences

    Defaults work for most teams.
  </Step>

  <Step title="Test Integration">
    Verify connection:
    ```
    "Show me my recent HubSpot contacts"
    "What deals are in the pipeline?"
    ```

    <Check>
      HubSpot connected! Ready for marketing and sales automation.
    </Check>
  </Step>
</Steps>

## Example Workflows

<AccordionGroup>
  <Accordion title="Lead Capture to Deal" icon="arrow-right">
    **Complete Lead Flow**:
    ```
    Trigger: Form submitted on website
    
    Workflow:
    1. Create contact in HubSpot
    2. Enrich with company data
    3. Calculate lead score
    4. If score > 70:
       - Create deal automatically
       - Assign to sales rep
       - Send notification to rep
    5. If score < 70:
       - Add to nurture workflow
       - Schedule follow-up email sequence
    6. Log all activities
    7. Notify team in Slack
    ```
  </Accordion>

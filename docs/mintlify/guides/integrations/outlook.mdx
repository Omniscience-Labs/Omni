## Setup

<Steps>
  <Step title="Connect Outlook">
    In Integrations, click **Connect Outlook**.

    {/* 
      ðŸ“¸ MEDIA PLACEHOLDER: Outlook Connection
      - Outlook/Microsoft 365 integration card
      - Connect button prominent
      - Capabilities listed
      - Size: 1200x800px
    */}
  </Step>

  <Step title="Microsoft OAuth">
    Sign in with Microsoft:
    - Choose Microsoft account (work or personal)
    - Select organization if applicable
    - Review requested permissions
    - Click **Accept**

    <Info>
      Omni requests permissions for: Mail (read/send), Calendar (read/write), Contacts (read), Tasks (read/write).
    </Info>
  </Step>

  <Step title="Configure Mail Settings">
    **Optional Setup**:
    - Default signature handling
    - Folder organization rules
    - Auto-reply settings
    - Notification preferences

    {/* 
      ðŸ“¸ MEDIA PLACEHOLDER: Outlook Settings
      - Configuration panel
      - Toggle options for features
      - Folder selector
      - Signature preview
      - Size: 1400x800px
    */}
  </Step>

  <Step title="Test Connection">
    Verify integration:
    ```
    "Check my Outlook inbox for unread messages"
    "Show my calendar for this week"
    ```

    <Check>
      Outlook connected! Enterprise email automation enabled.
    </Check>
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Authentication Issues" icon="key">
    **Common Fixes**:
    - Re-authenticate through Microsoft
    - Check admin consent given
    - Verify account has mail license
    - Ensure correct tenant selected

    Enterprise accounts may require admin approval.
  </Accordion>

  <Accordion title="Calendar Not Syncing" icon="calendar-xmark">
    **Check**:
    - Calendar permissions granted
    - Correct calendar selected
    - Time zone settings match
    - No sync conflicts

    Refresh connection if needed.
  </Accordion>
</AccordionGroup>

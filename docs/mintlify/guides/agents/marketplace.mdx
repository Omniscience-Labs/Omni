## Installing Agents

<Steps>
  <Step title="Click Install">
    On the agent detail page, click **Install Agent**.

    <Info>
      Installation is instant - no downloads or setup required.
    </Info>
  </Step>

  <Step title="Customize (Optional)">
    Personalize the agent:
    - **Name**: Keep default or rename
    - **Icon/Color**: Change visual identity
    - **Modifications**: Adjust system prompt if needed

    {/* 
      📸 MEDIA PLACEHOLDER: Installation Dialog
      - Agent preview with current config
      - Customization options
      - "Keep defaults" checkbox
      - Install and Cancel buttons
      - Size: 1200x800px
    */}

    <Tip>
      Start with default settings, then customize after testing.
    </Tip>
  </Step>

  <Step title="Grant Integration Access">
    If the agent requires integrations:
    1. Review requested permissions
    2. Connect required services
    3. Authorize access

    <Warning>
      Skip integration setup only if you're sure you won't use those features.
    </Warning>
  </Step>

  <Step title="Start Using">
    The agent appears in your workspace immediately:
    - Find it in your Agents list
    - Click to start a conversation
    - Test with a typical task

    <Check>
      Agent successfully installed and ready to use!
    </Check>
  </Step>
</Steps>

## Publishing Your Agents

Share your custom agents with the community.

### Preparation

<AccordionGroup>
  <Accordion title="Polish Your Agent" icon="sparkles">
    Before publishing:
    - ✓ Clear, descriptive name and description
    - ✓ Well-crafted system prompt
    - ✓ Optimal tool configuration
    - ✓ Tested with various tasks
    - ✓ Documentation of use cases
  </Accordion>

  <Accordion title="Create Examples" icon="lightbulb">
    Help users understand your agent:
    - Provide 3-5 sample tasks
    - Show expected outputs
    - Include edge cases
    - Demonstrate key features
  </Accordion>

  <Accordion title="Write Clear Documentation" icon="book">
    Essential information:
    - What problems it solves
    - Who should use it
    - How to get started
    - Tips for best results
    - Known limitations
  </Accordion>
</AccordionGroup>

### Publishing Process

<Steps>
  <Step title="Initiate Publication">
    In agent settings, click **Publish to Marketplace**.

    {/* 
      📸 MEDIA PLACEHOLDER: Publish Dialog
      - Publication form with required fields
      - Category selection dropdown
      - Tags input field
      - Preview mode toggle
      - Size: 1400x900px
    */}
  </Step>

  <Step title="Complete Listing Information">
    **Required Fields**:
    - **Display Name**: Public-facing name
    - **Short Description**: One-line summary (120 chars)
    - **Full Description**: Detailed explanation
    - **Category**: Primary classification
    - **Tags**: Searchable keywords
    - **Use Cases**: 3-5 example scenarios

    **Optional Fields**:
    - **Video Demo**: YouTube or Loom link
    - **Documentation**: External guide link
    - **Support Contact**: For user questions
  </Step>

  <Step title="Set Availability">
    Choose visibility:
    - **Public**: Anyone can install
    - **Unlisted**: Only those with link
    - **Team Only**: Your organization members

    <Note>
      Start with "Unlisted" to gather feedback before going public.
    </Note>
  </Step>

  <Step title="Review and Submit">
    Preview how your agent appears in the marketplace:
    - Check all information accuracy
    - Test sample conversations
    - Verify integration requirements
    - Confirm license terms

    Submit for review (approval typically within 24 hours).
  </Step>
</Steps>

## Rating and Reviews

### Leaving Reviews

Help others by reviewing installed agents:

<Steps>
  <Step title="Use the Agent">
    Test thoroughly across multiple use cases before reviewing.
  </Step>

  <Step title="Rate Performance">
    Consider:
    - **Accuracy**: Does it produce correct results?
    - **Reliability**: Consistent performance?
    - **Usability**: Easy to work with?
    - **Documentation**: Clear instructions?
    - **Value**: Solves real problems?
  </Step>

  <Step title="Write Detailed Feedback">
    **Good Review Template**:
    ```
    [Star Rating]: ⭐⭐⭐⭐⭐

    What I use it for:
    [Your use case]

    What works well:
    - [Strength 1]
    - [Strength 2]

    What could improve:
    - [Suggestion 1]
    - [Suggestion 2]

    Would recommend for:
    [Target users/use cases]
    ```
  </Step>
</Steps>

<Tip>
  **Be Constructive**: Even in critical reviews, offer specific suggestions for improvement rather than just listing problems.
</Tip>

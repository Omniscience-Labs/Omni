## Data Analysis Workflow

<Steps>
  <Step title="Upload Data">

    Agent automatically performs:

## Data Cleaning

<Steps>
  <Step title="Identify Issues">
    Agent automatically detects:
    - Missing values
    - Duplicate rows
    - Inconsistent formatting
    - Outliers
    - Data type mismatches

    ```
    "Analyze data quality and report issues"
    ```
  </Step>

  <Step title="Apply Cleaning Rules">
    ```
    "Clean this dataset:
    - Remove exact duplicate rows
    - Fill missing prices with category average
    - Standardize dates to YYYY-MM-DD
    - Remove rows with invalid emails
    - Trim whitespace from all text fields"
    ```

    Explicit rules for transparency.
  </Step>

  <Step title="Verify Cleaning">
    Agent provides:
    - Before/after comparison
    - Changes summary
    - Data quality metrics
    - Rows affected count

## Best Practices

<AccordionGroup>
  <Accordion title="Start with Exploration" icon="compass">
    **Always Begin With**:
    ```
    "Show me the first 10 rows and describe the data"
    ```

    Understand before analyzing.
  </Accordion>

  <Accordion title="Incremental Analysis" icon="stairs">
    **Build Up Gradually**:
    1. Basic statistics
    2. Simple visualizations
    3. Deeper analysis
    4. Complex modeling

    Don't jump to advanced analysis immediately.
  </Accordion>

  <Accordion title="Save Intermediate Results" icon="floppy-disk">
    **Create Checkpoints**:
    - Cleaned data
    - Calculated features
    - Generated visualizations
    - Analysis results

    Easy to iterate without reprocessing.
  </Accordion>
</AccordionGroup>

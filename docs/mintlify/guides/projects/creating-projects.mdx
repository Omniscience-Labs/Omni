## Quick Project Creation

<Steps>
  <Step title="Click New Project">
    From the sidebar or Projects page, click **New Project** or use `Cmd/Ctrl + Shift + N`.

    {/* 
      ðŸ“¸ MEDIA PLACEHOLDER: New Project Button
      - Highlighted button location in UI
      - Keyboard shortcut reminder visible
      - Size: 800x600px
    */}
  </Step>

  <Step title="Enter Basic Details">
    **Required**:
    - **Name**: Clear, descriptive project title
    - **Description**: Brief overview of project purpose

    **Optional**:
    - Icon (choose from 100+ options)
    - Color (for visual organization)

    <Tip>
      Choose distinctive colors and icons to quickly identify projects at a glance.
    </Tip>
  </Step>

  <Step title="Configure Settings">
    **Initial Setup**:
    - Default agent (optional)
    - Team members (add later if needed)
    - Knowledge base access

    {/* 
      ðŸ“¸ MEDIA PLACEHOLDER: Project Creation Modal
      - Form showing all fields
      - Icon selector grid
      - Color picker interface
      - Team invitation section
      - Size: 1600x1000px
    */}
  </Step>

  <Step title="Create and Start">
    Click **Create Project** and you'll be taken to your first thread.

    <Check>
      Project created! Ready for your first conversation.
    </Check>
  </Step>
</Steps>

## Advanced Configuration

<AccordionGroup>
  <Accordion title="Default Agent Assignment" icon="robot">
    Set which agent starts conversations automatically:
    - Saves time on every new thread
    - Ensures consistency
    - Can override per-thread
    - Update anytime in settings
  </Accordion>

  <Accordion title="Knowledge Base Integration" icon="book">
    Connect relevant documents:
    - **Global**: Company-wide knowledge
    - **Project-specific**: Dedicated files
    - **Inherited**: From parent folders

    Files accessible to all project threads.
  </Accordion>

  <Accordion title="Auto-Archive Rules" icon="box-archive">
    Automatically archive when:
    - Project status set to complete
    - No activity for X days
    - Specific date reached
    - Manual trigger only

    Keeps workspace organized.
  </Accordion>
</AccordionGroup>

## Project Naming Strategies

<CardGroup cols={2}>
  <Card title="By Client" icon="building">
    `[Client Name] - [Engagement Type]`
    
    Examples:
    - "Acme Corp - Website Redesign"
    - "TechStart - Q1 Consulting"
  </Card>
  <Card title="By Campaign" icon="bullhorn">
    `[Campaign Name] - [Time Period]`
    
    Examples:
    - "Product Launch - Fall 2024"
    - "Holiday Campaign - Q4"
  </Card>
  <Card title="By Product" icon="box">
    `[Product] - [Initiative]`
    
    Examples:
    - "Mobile App - v2.0 Redesign"
    - "API Platform - Documentation"
  </Card>
  <Card title="By Department" icon="users">
    `[Team] - [Focus Area]`
    
    Examples:
    - "Sales - Enablement Materials"
    - "HR - Onboarding Process"
  </Card>
</CardGroup>

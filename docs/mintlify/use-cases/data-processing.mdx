## Automated Reporting

**Weekly Sales Report Workflow**:

<Steps>
  <Step title="Schedule">
    ```
    Trigger: Every Monday at 8 AM
    ```
  </Step>

  <Step title="Data Collection">
    ```
    Pull data from:
    - CRM API (sales closed last week)
    - Google Sheets (pipeline data)
    - Analytics platform (website metrics)
    ```
  </Step>

  <Step title="Processing">
    ```
    Calculate:
    - Week-over-week changes
    - Top performers
    - Pipeline value
    - Conversion rates
    ```
  </Step>

  <Step title="Visualization">
    ```
    Create:
    - Summary slide deck
    - Detailed charts
    - Comparison tables
    ```
  </Step>

  <Step title="Distribution">
    ```
    Send:
    - PDF to sales-leadership@company.com
    - Post summary to #sales Slack channel
    - Save to Reports folder in Drive
    ```

    <Check>
      Automated end-to-end reporting!
    </Check>
  </Step>

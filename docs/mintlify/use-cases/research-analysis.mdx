## Complete Research Workflow

<Steps>
  <Step title="Define Research Question">
    Start with clear objective:
    ```
    Clear: "What are the top 3 emerging trends in 
    sustainable packaging for food products?"
    
    Vague: "Research sustainability stuff"
    ```

    Specificity drives quality.
  </Step>

  <Step title="Agent Plans Approach">
    Agent outlines research methodology:
    ```
    Research Plan:
    1. Web search for recent articles (2023-2024)
    2. Identify authoritative sources
    3. Extract key statistics and trends
    4. Cross-reference findings
    5. Synthesize into report
    ```

    Review and adjust before execution.

    {/* 
      üì∏ MEDIA PLACEHOLDER: Research Plan
      - Structured plan with steps
      - Edit/approve buttons
      - Estimated time
      - Sources to check listed
      - Size: 1400x800px
    */}
  </Step>

  <Step title="Information Gathering">
    Agent collects data:
    - Web searches across multiple queries
    - Visits and scrapes relevant pages
    - Extracts quantitative data
    - Gathers expert quotes
    - Documents all sources

    {/* 
      üé• VIDEO PLACEHOLDER: Research in Progress (35 seconds)
      - Agent performing multiple searches
      - Visiting and extracting from sites
      - Organizing findings

  </Step>

  <Step title="Analysis & Synthesis">
    Agent processes findings:
    - Identifies patterns and themes
    - Calculates statistics
    - Creates visualizations
    - Draws conclusions
    - Formulates recommendations

    Deep analysis beyond summarization.
  </Step>

  <Step title="Report Generation">
    Professional deliverable:
    ```
    Report Structure:
    - Executive Summary (1 page)
    - Methodology
    - Findings with visualizations
    - Analysis and insights
    - Recommendations
    - Detailed bibliography
    ```

    <Check>
      Research complete with fully cited, publication-ready report!
    </Check>
  </Step>

  <Step title="Follow-Up Questions">
    Dig deeper:
    ```
    "Can you expand on trend #2 with more examples?"
    "Add a section on regulatory implications"
    "Create a presentation version of this"
    ```

    Iterate until perfect.
  </Step>
</Steps>

## Research Best Practices

<AccordionGroup>
  <Accordion title="Source Quality" icon="award">
    **Prioritize Authoritative Sources**:
    - Peer-reviewed journals
    - Government data
    - Industry reports from known firms
    - Company financial filings

    Cross-reference claims across multiple sources.
  </Accordion>

  <Accordion title="Recency" icon="clock">
    **Use Current Information**:
    ```
    "Focus on sources from the last 12 months"
    "Prioritize 2024 data over historical"
    "Note publication dates for all statistics"
    ```

    Especially critical for technology and markets.
  </Accordion>

  <Accordion title="Citation Standards" icon="quote-right">
    **Proper Attribution**:
    - Include URLs for all sources
    - Note access dates
    - Cite specific pages/sections
    - Use consistent citation format

    Maintains credibility and traceability.
  </Accordion>

  <Accordion title="Verify Claims" icon="check-double">
    **Cross-Check Facts**:
    ```
    "Verify this statistic across multiple sources"
    "Find contradictory information if it exists"
    "Note confidence level for each claim"
    ```

    Don't rely on single sources for critical facts.
  </Accordion>
</AccordionGroup>

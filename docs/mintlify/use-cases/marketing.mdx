## SEO Workflows

<AccordionGroup>
  <Accordion title="Keyword Research" icon="key">
    ```
    "Research keywords for {{topic}}:
    
    Analysis:
    - Search volume trends
    - Competition level
    - Keyword difficulty
    - Related keywords
    - Question-based queries
    - Long-tail opportunities
    
    Prioritization:
    - High volume, low competition
    - Intent alignment
    - Conversion potential
    
    Deliverable: Keyword strategy spreadsheet"
    ```
  </Accordion>

  <Accordion title="Content Optimization" icon="magnifying-glass-plus">
    ```
    "Optimize blog post for SEO:
    
    - Target keyword: {{primary_keyword}}
    - Current ranking: Not ranking
    
    Tasks:
    - Add keyword naturally (2-3% density)
    - Optimize title tag (include keyword)
    - Write compelling meta description
    - Improve header structure
    - Add internal links to related content
    - Optimize images (alt text, file names)
    - Add schema markup suggestions
    
    Don't compromise readability for SEO"
    ```
  </Accordion>

  <Accordion title="Competitor Content Analysis" icon="users-viewfinder">
    ```
    "Analyze competitor content for {{keyword}}:
    
    For top 10 ranking pages:
    - Word count
    - Content structure
    - Topics covered
    - Multimedia usage
    - Backlink profile
    
    Recommendations:
    - Content gaps we can fill
    - Better angles to take
    - Opportunities they missed"
    ```
  </Accordion>
</AccordionGroup>

    ```
    Trigger: Lead downloads ebook
    
    Follow-up Sequence:
    Day 0: Thank you + resource delivery
    Day 2: Related content + value add
    Day 5: Case study + social proof
    Day 7: Demo invitation
    Day 10: Personalized outreach
    Day 14: Final touch + alternatives

    ```

    4. Offer incentive to return
    5. Survey if no response
    6. Archive if still no response
